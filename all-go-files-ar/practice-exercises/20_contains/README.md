# شرح التمرين `20_contains` — contains

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `Contains(nums []int, target int) bool`

## `exercise.go`

```go
package main

func Contains(nums []int, target int) bool {
	for _, n := range nums {
		if n == target {
			return true
		}
	}
	return false
}
```

### شرح كل سطر في الحل

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package main` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func Contains(nums []int, target int) bool {` | تعريف دالة `Contains` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `for _, n := range nums {` | حلقة `range` تمر على عناصر النص أو الـslice؛ المتغير `_` يعني تجاهل قيمة لا نحتاجها. |
| 5 | `if n == target {` | شرط `if`: ينفذ الكتلة التالية فقط عندما يكون التعبير المنطقي صحيحًا. |
| 6 | `return true` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 7 | `}` | نهاية الكتلة الحالية. |
| 8 | `}` | نهاية الكتلة الحالية. |
| 9 | `return false` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 10 | `}` | نهاية الكتلة الحالية. |

## كيف يعمل الحل؟

- يستخدم حلقة `for` لتكرار الفحص أو المعالجة.
- يستخدم `range` للمرور على عناصر slice أو أحرف string.

### تتبع عام للتنفيذ

1. يبدأ التنفيذ عند استدعاء الدالة الأساسية بالقيم المطلوبة.
2. تُنشأ المتغيرات المساعدة والقيم الابتدائية.
3. تنفذ الحلقات أو الشروط بالترتيب الظاهر في الكود.
4. عند تحقق حالة إرجاع مبكر تتوقف الدالة فورًا؛ وإلا تستمر حتى النتيجة النهائية.
5. القيمة المعادة أو التعديل النهائي يظهر في `main.go`.

## `main.go` المستخدم للتجربة

```go
package main

import "fmt"

func main() {
	fmt.Println(Contains([]int{1, 2, 3}, 2))
	fmt.Println(Contains([]int{1, 2, 3}, 9))
}
```
