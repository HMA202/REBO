# شرح التمرين `27_countwords` — countwords

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `CountWords(s string) int`

## `exercise.go`

```go
package main

func CountWords(s string) int {
	count := 0
	inWord := false
	for _, ch := range s {
		if ch == ' ' || ch == '\n' || ch == '\t' {
			inWord = false
		} else if !inWord {
			count++
			inWord = true
		}
	}
	return count
}
```

## كيف يعمل الحل؟

- يستخدم حلقة `for` لتكرار الفحص أو المعالجة.
- يستخدم `range` للمرور على عناصر slice أو أحرف string.
