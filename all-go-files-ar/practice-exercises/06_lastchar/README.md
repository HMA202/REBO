# شرح التمرين `06_lastchar` — lastchar

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `LastChar(s string) rune`

## `exercise.go`

```go
package main

func LastChar(s string) rune {
	last := rune(0)
	for _, ch := range s {
		last = ch
	}
	return last
}
```

### شرح كل سطر في الحل

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package main` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func LastChar(s string) rune {` | تعريف دالة `LastChar` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `last := rune(0)` | إنشاء المتغير `last` لأول مرة واستنتاج نوعه تلقائيًا من القيمة الموجودة يمين `:=`. |
| 5 | `for _, ch := range s {` | حلقة `range` تمر على عناصر النص أو الـslice؛ المتغير `_` يعني تجاهل قيمة لا نحتاجها. |
| 6 | `last = ch` | إسناد أو تحديث قيمة `last` وفق التعبير الموجود في الجهة اليمنى. |
| 7 | `}` | نهاية الكتلة الحالية. |
| 8 | `return last` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 9 | `}` | نهاية الكتلة الحالية. |

## كيف يعمل الحل؟

- يستخدم حلقة `for` لتكرار الفحص أو المعالجة.
- يستخدم `range` للمرور على عناصر slice أو أحرف string.
- يحتوي استدعاءات دوال؛ راقب هل يوجد recursion أو دوال مساعدة.

### تتبع عام للتنفيذ

1. يبدأ التنفيذ عند استدعاء الدالة الأساسية بالقيم المطلوبة.
2. تُنشأ المتغيرات المساعدة والقيم الابتدائية.
3. تنفذ الحلقات أو الشروط بالترتيب الظاهر في الكود.
4. عند تحقق حالة إرجاع مبكر تتوقف الدالة فورًا؛ وإلا تستمر حتى النتيجة النهائية.
5. القيمة المعادة أو التعديل النهائي يظهر في `main.go`.

## `main.go` المستخدم للتجربة

```go

import "fmt"

func main() {
	fmt.Printf("%c\n", LastChar("hello"))
	fmt.Println(LastChar(""))
```
