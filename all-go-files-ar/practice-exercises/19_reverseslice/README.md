# شرح التمرين `19_reverseslice` — reverseslice

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `ReverseSlice(nums []int) []int`

## `exercise.go`

```go
package main

func ReverseSlice(nums []int) []int {
	result := make([]int, len(nums))
	for i := 0; i < len(nums); i++ {
		result[i] = nums[len(nums)-1-i]
	}
	return result
}
```

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package main` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func ReverseSlice(nums []int) []int {` | تعريف دالة `ReverseSlice` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `result := make([]int, len(nums))` | ينشئ slice أو بنية ديناميكية بالحجم المطلوب ويخزنها في متغير جديد. |
| 5 | `for i := 0; i < len(nums); i++ {` | حلقة `for` بعداد أو شرط؛ تتحكم في تكرار الأوامر الموجودة داخلها. |
| 6 | `result[i] = nums[len(nums)-1-i]` | إسناد أو تحديث قيمة `result[i]` وفق التعبير الموجود في الجهة اليمنى. |
| 7 | `}` | نهاية الكتلة الحالية. |
| 8 | `return result` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 9 | `}` | نهاية الكتلة الحالية. |

## كيف يعمل الحل؟

- يستخدم حلقة `for` لتكرار الفحص أو المعالجة.
- يحتوي استدعاءات دوال؛ راقب هل يوجد recursion أو دوال مساعدة.

### تتبع عام للتنفيذ

1. يبدأ التنفيذ عند استدعاء الدالة الأساسية بالقيم المطلوبة.
2. تُنشأ المتغيرات المساعدة والقيم الابتدائية.
3. تنفذ الحلقات أو الشروط بالترتيب الظاهر في الكود.
4. عند تحقق حالة إرجاع مبكر تتوقف الدالة فورًا؛ وإلا تستمر حتى النتيجة النهائية.
5. القيمة المعادة أو التعديل النهائي يظهر في `main.go`.
