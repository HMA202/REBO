# شرح التمرين `43_sum_digits_in_string` — sum digits in string

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `SumDigitsInString(s string) int`

## `exercise.go`

```go
package main

func SumDigitsInString(s string) int {
	sum := 0

	for i := 0; i < len(s); i++ {
		if s[i] >= '0' && s[i] <= '9' {
			sum += int(s[i] - '0')
		}
	}

	return sum
}
```
