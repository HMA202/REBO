# شرح التمرين `57_longest_unique_substring` — longest unique substring

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `LongestUniqueSubstring(s string) int`

## `exercise.go`

```go
package main

func LongestUniqueSubstring(s string) int {
	lastSeen := [256]int{}

	for i := 0; i < 256; i++ {
		lastSeen[i] = -1
	}

	start := 0
	best := 0

	for i := 0; i < len(s); i++ {
		ch := s[i]

		if lastSeen[ch] >= start {
			start = lastSeen[ch] + 1
		}

		lastSeen[ch] = i

		length := i - start + 1
		if length > best {
			best = length
		}
	}

	return best
}
```
