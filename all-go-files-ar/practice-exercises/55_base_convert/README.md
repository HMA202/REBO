# شرح التمرين `55_base_convert` — base convert

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `digitValue(ch byte) int`
- `digitChar(n int) byte`
- `BaseConvert(s string, fromBase int, toBase int) string`

## `exercise.go`

```go
package main

func digitValue(ch byte) int {
	if ch >= '0' && ch <= '9' {
		return int(ch - '0')
	}
	if ch >= 'A' && ch <= 'F' {
		return int(ch-'A') + 10
	}
	if ch >= 'a' && ch <= 'f' {
		return int(ch-'a') + 10
	}

	return -1
}

func digitChar(n int) byte {
	if n < 10 {
		return byte('0' + n)
	}
	return byte('A' + n - 10)
}

func BaseConvert(s string, fromBase int, toBase int) string {
	if fromBase < 2 || fromBase > 16 || toBase < 2 || toBase > 16 {
		return ""
	}

	decimal := 0

	for i := 0; i < len(s); i++ {
		value := digitValue(s[i])
		if value < 0 || value >= fromBase {
			return ""
		}

		decimal = decimal*fromBase + value
	}

	if decimal == 0 {
		return "0"
	}

	result := ""

	for decimal > 0 {
		result = string(digitChar(decimal%toBase)) + result
		decimal /= toBase
	}

	return result
}
```

## كيف يعمل الحل؟

- يستخدم حلقة `for` لتكرار الفحص أو المعالجة.
- يحتوي استدعاءات دوال؛ راقب هل يوجد recursion أو دوال مساعدة.
- عامل `%` يُستخدم للباقي، وغالبًا لفحص القابلية للقسمة أو استخراج الخانات.
- قد يستخدم مؤشرات أو عمليات ضرب؛ يُحدد المعنى من السياق والتوقيع.

### تتبع عام للتنفيذ

1. يبدأ التنفيذ عند استدعاء الدالة الأساسية بالقيم المطلوبة.
2. تُنشأ المتغيرات المساعدة والقيم الابتدائية.
3. تنفذ الحلقات أو الشروط بالترتيب الظاهر في الكود.
4. عند تحقق حالة إرجاع مبكر تتوقف الدالة فورًا؛ وإلا تستمر حتى النتيجة النهائية.
5. القيمة المعادة أو التعديل النهائي يظهر في `main.go`.
