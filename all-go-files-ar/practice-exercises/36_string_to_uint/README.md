# شرح التمرين `36_string_to_uint` — string to uint

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `StringToUint(s string) uint`

## `exercise.go`

```go
package main

func StringToUint(s string) uint {
	var result uint = 0

	if s == "" {
		return 0
	}

	for i := 0; i < len(s); i++ {
		if s[i] < '0' || s[i] > '9' {
			return 0
		}

		result = result*10 + uint(s[i]-'0')
	}

	return result
}
```
