# شرح التمرين `52_csv_parse_line` — csv parse line

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `CSVParseLine(line string) []string`

## `exercise.go`

```go
package main

func CSVParseLine(line string) []string {
	result := []string{}
	current := ""
	inQuotes := false

	for i := 0; i < len(line); i++ {
		ch := line[i]

		if ch == '"' {
			inQuotes = !inQuotes
		} else if ch == ',' && !inQuotes {
			result = append(result, current)
			current = ""
		} else {
			current += string(ch)
		}
	}

	result = append(result, current)

	return result
}
```
