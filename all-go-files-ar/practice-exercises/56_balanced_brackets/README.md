# شرح التمرين `56_balanced_brackets` — balanced brackets

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `BalancedBrackets(s string) bool`

## `exercise.go`

```go
package main

func BalancedBrackets(s string) bool {
	stack := []rune{}

	for _, ch := range s {
		if ch == '(' || ch == '[' || ch == '{' {
			stack = append(stack, ch)
		} else if ch == ')' || ch == ']' || ch == '}' {
			if len(stack) == 0 {
				return false
			}

			last := stack[len(stack)-1]
			stack = stack[:len(stack)-1]

			if ch == ')' && last != '(' {
				return false
			}
			if ch == ']' && last != '[' {
				return false
			}
			if ch == '}' && last != '{' {
				return false
			}
		}
	}

	return len(stack) == 0
}
```

## كيف يعمل الحل؟

- يستخدم حلقة `for` لتكرار الفحص أو المعالجة.
- يستخدم `range` للمرور على عناصر slice أو أحرف string.
- يحتوي استدعاءات دوال؛ راقب هل يوجد recursion أو دوال مساعدة.
- يستخدم `append` لبناء slice ديناميكيًا.
- قد يستخدم مؤشرات أو عمليات ضرب؛ يُحدد المعنى من السياق والتوقيع.

### تتبع عام للتنفيذ

1. يبدأ التنفيذ عند استدعاء الدالة الأساسية بالقيم المطلوبة.
2. تُنشأ المتغيرات المساعدة والقيم الابتدائية.
3. تنفذ الحلقات أو الشروط بالترتيب الظاهر في الكود.
4. عند تحقق حالة إرجاع مبكر تتوقف الدالة فورًا؛ وإلا تستمر حتى النتيجة النهائية.
5. القيمة المعادة أو التعديل النهائي يظهر في `main.go`.

## `main.go` المستخدم للتجربة

```go
package main

import "fmt"

func main() {
	fmt.Println(BalancedBrackets("([]{})"))
	fmt.Println(BalancedBrackets("([)]"))
	fmt.Println(BalancedBrackets("((hello))"))
}
```
