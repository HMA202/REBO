# شرح التمرين `44_count_upper_lower` — count upper lower

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `CountUpperLower(s string) (int, int)`

## `exercise.go`

```go
package main

func CountUpperLower(s string) (int, int) {
	upper := 0
	lower := 0

	for _, ch := range s {
		if ch >= 'A' && ch <= 'Z' {
			upper++
		} else if ch >= 'a' && ch <= 'z' {
			lower++
		}
	}

	return upper, lower
}
```
