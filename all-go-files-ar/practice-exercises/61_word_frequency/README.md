# شرح التمرين `61_word_frequency` — word frequency

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `WordFrequency(s string) map[string]int`

## `exercise.go`

```go
package main

func WordFrequency(s string) map[string]int {
	frequency := map[string]int{}
	word := ""

	for i := 0; i <= len(s); i++ {
		ch := byte(' ')

		if i < len(s) {
			ch = s[i]
		}

		if ch == ' ' {
			if word != "" {
				frequency[word]++
				word = ""
			}
		} else {
			word += string(ch)
		}
	}

	return frequency
}
```
