# شرح التمرين `64_caesar_cipher` — caesar cipher

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `CaesarCipher(s string, shift int) string`

## `exercise.go`

```go
package main

func CaesarCipher(s string, shift int) string {
	result := ""
	shift = shift % 26

	for _, ch := range s {
		if ch >= 'a' && ch <= 'z' {
			offset := int(ch - 'a')
			result += string(rune('a' + (offset+shift+26)%26))
		} else if ch >= 'A' && ch <= 'Z' {
			offset := int(ch - 'A')
			result += string(rune('A' + (offset+shift+26)%26))
		} else {
			result += string(ch)
		}
	}

	return result
}
```

## كيف يعمل الحل؟

- يستخدم حلقة `for` لتكرار الفحص أو المعالجة.
- يستخدم `range` للمرور على عناصر slice أو أحرف string.
- يحتوي استدعاءات دوال؛ راقب هل يوجد recursion أو دوال مساعدة.
- عامل `%` يُستخدم للباقي، وغالبًا لفحص القابلية للقسمة أو استخراج الخانات.
- قد يستخدم مؤشرات أو عمليات ضرب؛ يُحدد المعنى من السياق والتوقيع.
