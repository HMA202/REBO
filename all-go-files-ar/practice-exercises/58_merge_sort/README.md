# شرح التمرين `58_merge_sort` — merge sort

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `merge(left []int, right []int) []int`
- `MergeSort(nums []int) []int`

## `exercise.go`

```go
package main

func merge(left []int, right []int) []int {
	result := []int{}
	i := 0
	j := 0

	for i < len(left) && j < len(right) {
		if left[i] <= right[j] {
			result = append(result, left[i])
			i++
		} else {
			result = append(result, right[j])
			j++
		}
	}

	for i < len(left) {
		result = append(result, left[i])
		i++
	}

	for j < len(right) {
		result = append(result, right[j])
		j++
	}

	return result
}

func MergeSort(nums []int) []int {
	if len(nums) <= 1 {
		copyNums := make([]int, len(nums))
		copy(copyNums, nums)
		return copyNums
	}

	mid := len(nums) / 2

	left := MergeSort(nums[:mid])
	right := MergeSort(nums[mid:])

	return merge(left, right)
}
```

- قد يستخدم مؤشرات أو عمليات ضرب؛ يُحدد المعنى من السياق والتوقيع.
