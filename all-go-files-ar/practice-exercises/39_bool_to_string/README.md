# شرح التمرين `39_bool_to_string` — bool to string

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `BoolToString(value bool) string`

## `exercise.go`

```go
package main

func BoolToString(value bool) string {
	if value {
		return "true"
	}

	return "false"
}
```

| السطر | التعليمة | الشرح |

| 1 | `package main` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func BoolToString(value bool) string {` | تعريف دالة `BoolToString` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |

| 5 | `return "true"` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 6 | `}` | نهاية الكتلة الحالية. |
