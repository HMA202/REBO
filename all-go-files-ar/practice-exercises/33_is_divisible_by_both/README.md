# شرح التمرين `33_is_divisible_by_both` — is divisible by both

## فكرة المجلد

هذا تمرين تدريبي إضافي. ملف `exercise.go` يحتوي الحل أو الدوال الأساسية، وملف `main.go` يحتوي أمثلة تشغيل تساعد على رؤية النتيجة.

## الدوال الموجودة

- `IsDivisibleByBoth(n, a, b uint) bool`

## `exercise.go`

```go
package main

func IsDivisibleByBoth(n, a, b uint) bool {
	if a == 0 || b == 0 {
		return false
	}

	return n%a == 0 && n%b == 0
}
```
