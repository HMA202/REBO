# شرح برنامج `printalphabet` — طباعة الأبجدية الإنجليزية من a إلى z

## الكود

```go
package main

import "github.com/01-edu/z01"

func main() {
	for r := 'a'; r <= 'z'; r++ {
		z01.PrintRune(r)
	}
	z01.PrintRune('\n')
}
```

## المطلوب

هذا ملف برنامج كامل لأن الحزمة هي `package main` وفيه دالة `main()` التي يبدأ منها التشغيل.

## تدفق التنفيذ

1. بدء rune بالحرف a.
2. زيادة rune حتى z.
3. طباعة كل حرف ثم newline.

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package main` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `import "github.com/01-edu/z01"` | يستورد الحزمة المذكورة حتى يمكن استخدام دوالها أو قيمها داخل الملف. |
| 5 | `func main() {` | تعريف دالة `main` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 6 | `for r := 'a'; r <= 'z'; r++ {` | حلقة `for` بعداد أو شرط؛ تتحكم في تكرار الأوامر الموجودة داخلها. |
| 7 | `z01.PrintRune(r)` | يطبع rune واحدة باستخدام الدالة المسموحة في تمارين piscine. |
| 8 | `}` | نهاية الكتلة الحالية. |
| 9 | `z01.PrintRune('\n')` | يطبع rune واحدة باستخدام الدالة المسموحة في تمارين piscine. |
| 10 | `}` | نهاية الكتلة الحالية. |

## نقاط مهمة

- الحروف والأرقام مكتوبة كـ `rune` بين اقتباس مفرد.
- قيمة rune يمكن زيادتها أو إنقاصها لأن Unicode code point قيمة عددية.
- `z01.PrintRune('\n')` تنهي السطر.
