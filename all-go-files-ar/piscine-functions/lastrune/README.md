# شرح `lastrune.go` — إرجاع آخر rune

> الدالة الأساسية: `LastRune`

## ما وظيفة الكود؟

تمر على النص كاملًا وتحفظ آخر rune تمت رؤيتها، ثم تعيدها. النص الفارغ يعيد rune صفرية.

## الكود الأصلي

```go
package piscine

func LastRune(s string) rune {
	var last rune

	for _, r := range s {
		last = r
	}

	return last
}
```

## المدخلات والمخرجات

```go
func LastRune(s string) rune
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. تعريف last بقيمته الصفرية.
2. لكل rune، استبدال last بالحرف الحالي.
3. بعد انتهاء الحلقة تكون last هي آخر حرف.

## تتبع بأمثلة

- `LastRune("Hello")` ⟶ `'o'`
- `LastRune("أهلا")` ⟶ `'ا'`
- `LastRune("")` ⟶ `0`

## المفاهيم المهمة

- **rune وUTF-8**
- **القيمة الصفرية**

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package piscine` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func LastRune(s string) rune {` | تعريف دالة `LastRune` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `var last rune` | تعليمة Go ضمن الخوارزمية؛ معناها التفصيلي يتضح من قسم تدفق التنفيذ أعلاه ومن المتغيرات المحيطة بها. |
| 6 | `for _, r := range s {` | حلقة `range` تمر على عناصر النص أو الـslice؛ المتغير `_` يعني تجاهل قيمة لا نحتاجها. |
| 7 | `last = r` | إسناد أو تحديث قيمة `last` وفق التعبير الموجود في الجهة اليمنى. |
| 8 | `}` | نهاية الكتلة الحالية. |
| 10 | `return last` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 11 | `}` | نهاية الكتلة الحالية. |

## الحالات الخاصة والأخطاء الشائعة

- المرور ضروري هنا لأن آخر byte ليس بالضرورة آخر rune كاملة.

## التعقيد

الزمن O(n) والذاكرة O(1).
