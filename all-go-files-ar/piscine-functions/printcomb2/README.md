# شرح `printcomb2.go` — طباعة أزواج الأعداد من 00 إلى 99

> الدالة الأساسية: `PrintComb2`

## ما وظيفة الكود؟

تطبع جميع الأزواج 00 01 إلى 98 99 بحيث العدد الأول أصغر من الثاني، مع فواصل بين الأزواج.

## الكود الأصلي

```go
package piscine

import "github.com/01-edu/z01"

func PrintComb2() {
	for a1 := '0'; a1 <= '9'; a1++ {
		for a2 := '0'; a2 <= '9'; a2++ {
			for b1 := a1; b1 <= '9'; b1++ {
				for b2 := '0'; b2 <= '9'; b2++ {
					if b1 == a1 && b2 <= a2 {
						continue
					}

					z01.PrintRune(a1)
					z01.PrintRune(a2)
					z01.PrintRune(' ')
					z01.PrintRune(b1)
					z01.PrintRune(b2)

					if !(a1 == '9' && a2 == '8' && b1 == '9' && b2 == '9') {
						z01.PrintRune(',')
						z01.PrintRune(' ')
					}
				}
			}
		}
	}
	z01.PrintRune('\n')
}
```

## تدفق التنفيذ خطوة بخطوة

1. تمثيل كل عدد برقمين: a1,a2 وb1,b2.
2. السماح لـb1 أن يبدأ من a1 لتقليل البحث.
3. استخدام continue لتجاوز الحالات التي يكون فيها العدد الثاني أصغر أو مساويًا للأول.
4. طباعة الزوجين مع مسافة.
5. منع الفاصل بعد 98 99.

## تتبع بأمثلة

- `بداية الناتج` ⟶ `00 01, 00 02, ...`
- `نهاية الناتج` ⟶ `..., 97 99, 98 99`
