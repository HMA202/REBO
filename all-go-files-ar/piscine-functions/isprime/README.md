# شرح `isprime.go` — فحص العدد الأولي

> الدالة الأساسية: `IsPrime`

## ما وظيفة الكود؟

تحدد هل nb عدد أولي؛ تستبعد القيم الأقل من 2 والأعداد الزوجية، ثم تفحص القواسم الفردية فقط حتى الجذر.

## الكود الأصلي

```go
package piscine

func IsPrime(nb int) bool {
	if nb < 2 {
		return false
	}

	if nb == 2 {
		return true
	}

	if nb%2 == 0 {
		return false
	}

	for divisor := 3; divisor <= nb/divisor; divisor += 2 {
		if nb%divisor == 0 {
			return false
		}
	}

	return true
}
```

## المدخلات والمخرجات

```go
func IsPrime(nb int) bool
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. رفض كل nb أقل من 2.
2. قبول 2 مباشرة.
3. رفض الأعداد الزوجية الأخرى.
4. اختبار القواسم 3،5،7... ما دام divisor <= nb/divisor.
5. إرجاع false عند إيجاد قاسم وإلا true.

## تتبع بأمثلة

- `IsPrime(2)` ⟶ `true`
- `IsPrime(17)` ⟶ `true`
- `IsPrime(21)` ⟶ `false`

## المفاهيم المهمة

- **الأعداد الأولية**
- **عامل باقي القسمة %**
- **تحسين حتى الجذر**

## الحالات الخاصة والأخطاء الشائعة

- 1 و0 والسالب ليست أولية.
- صيغة nb/divisor تتجنب overflow المحتمل من divisor*divisor.

## التعقيد

الزمن O(√n) والذاكرة O(1).
