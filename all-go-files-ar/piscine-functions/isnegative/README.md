# شرح `isnegative.go` — طباعة هل العدد سالب

> الدالة الأساسية: `IsNegative`

## ما وظيفة الكود؟

تطبع T إذا كان العدد أقل من صفر، وإلا تطبع F، ثم سطرًا جديدًا.

## الكود الأصلي

```go
package piscine

import "github.com/01-edu/z01"

func IsNegative(nb int) {
	if nb < 0 {
		z01.PrintRune('T')
	} else {
		z01.PrintRune('F')
	}
	z01.PrintRune('\n')
}
```

## المدخلات والمخرجات

```go
func IsNegative(nb int)
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. مقارنة nb مع صفر.
2. طباعة T للحالة السالبة وإلا F.
3. طباعة newline.

## تتبع بأمثلة

- `IsNegative(-3)` ⟶ `T`
- `IsNegative(0)` ⟶ `F`
- `IsNegative(4)` ⟶ `F`

## المفاهيم المهمة

- **if/else**
- **z01.PrintRune**
