# شرح `pointone.go` — تعديل قيمة عبر pointer

> الدالة الأساسية: `PointOne`

## ما وظيفة الكود؟

تضع القيمة 1 داخل المتغير الذي يشير إليه المؤشر n.

## الكود الأصلي

```go
package piscine

func PointOne(n *int) {
	*n = 1
}
```

## المدخلات والمخرجات

```go
func PointOne(n *int)
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. استخدام *n للوصول إلى القيمة الأصلية.
2. إسناد 1 إلى تلك القيمة.

## تتبع بأمثلة

- `x=7; PointOne(&x)` ⟶ `x يصبح 1`

## المفاهيم المهمة

- **pointer**
- **عامل العنوان &**
- **فك الإشارة ***

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package piscine` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func PointOne(n *int) {` | تعريف دالة `PointOne` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `*n = 1` | يكتب قيمة جديدة داخل المتغير الأصلي عبر فك المؤشر باستخدام `*`. |
| 5 | `}` | نهاية الكتلة الحالية. |

## الحالات الخاصة والأخطاء الشائعة

- تمرير nil يؤدي إلى panic.

## التعقيد

O(1).
