# شرح `islower.go` — التحقق من الحروف الصغيرة

> الدالة الأساسية: `IsLower`

## ما وظيفة الكود؟

تعيد true إذا كانت كل محارف النص بين a وz فقط.

## الكود الأصلي

```go
package piscine

func IsLower(s string) bool {
	for _, r := range s {
		if r < 'a' || r > 'z' {
			return false
		}
	}

	return true
}
```

## المدخلات والمخرجات

```go
func IsLower(s string) bool
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. المرور على كل rune.
2. رفض أي rune أصغر من a أو أكبر من z.
3. إرجاع true إذا لم يوجد مخالف.

## تتبع بأمثلة

- `IsLower("hello")` ⟶ `true`
- `IsLower("Hello")` ⟶ `false`
- `IsLower("")` ⟶ `true`

## المفاهيم المهمة

- **مجال ASCII**
- **منطق OR**

## الحالات الخاصة والأخطاء الشائعة

- النص الفارغ يعيد true.
- الأرقام والمسافات تعطي false.

## التعقيد

الزمن O(n) والذاكرة O(1).
