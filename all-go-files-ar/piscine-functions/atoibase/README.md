# شرح `atoibase.go` — تحويل عدد مكتوب بقاعدة مخصصة إلى int

> الدالة الأساسية: `AtoiBase`

## ما وظيفة الكود؟

تحول النص s من قاعدة يحددها النص base إلى عدد عشري int، بعد التحقق من صلاحية القاعدة ومن أن كل رمز في العدد موجود داخل القاعدة.

## الكود الأصلي

```go
package piscine

func AtoiBase(s string, base string) int {
	baseRunes := []rune(base)

	if !validAtoiBase(baseRunes) {
		return 0
	}

	result := 0

	for _, r := range s {
		value := runePosition(r, baseRunes)

		if value == -1 {
			return 0
		}

		result = result*len(baseRunes) + value
	}

	return result
}

func validAtoiBase(base []rune) bool {
	if len(base) < 2 {
		return false
	}

	for i := 0; i < len(base); i++ {
		if base[i] == '+' || base[i] == '-' {
			return false
		}

		for j := i + 1; j < len(base); j++ {
			if base[i] == base[j] {
				return false
			}
		}
	}

	return true
}

func runePosition(r rune, base []rune) int {
	for i := 0; i < len(base); i++ {
		if r == base[i] {
			return i
		}
	}

	return -1
}
```

## تدفق التنفيذ خطوة بخطوة

1. تحويل القاعدة إلى []rune لدعم Unicode.
2. التحقق أن طول القاعدة اثنان على الأقل، ولا تحتوي + أو -، ولا يوجد حرف مكرر.
3. لكل حرف من s: البحث عن موقعه داخل القاعدة؛ موقع الحرف هو قيمته الرقمية.
4. تحديث الناتج بصيغة القيمة الموضعية: result = result*baseLength + value.
5. إرجاع 0 إذا كانت القاعدة أو أحد رموز العدد غير صالح.

## تتبع بأمثلة

- `AtoiBase("1011", "01")` ⟶ `11`
- `AtoiBase("ff", "0123456789abcdef")` ⟶ `255`
- `AtoiBase("ba", "abc")` ⟶ `3`

## الحالات الخاصة والأخطاء الشائعة

- قاعدة بطول أقل من 2 غير صالحة.
- تكرار أي رمز يجعل القاعدة غير صالحة.
- وجود + أو - في القاعدة غير مسموح.
- أي رمز من s غير موجود في base يؤدي إلى 0.

## التعقيد

إذا كان طول العدد n وطول القاعدة b فالزمن O(n×b)، والذاكرة O(b) بسبب []rune.
