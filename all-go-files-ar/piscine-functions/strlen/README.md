# شرح `strlen.go` — حساب عدد حروف Unicode

> الدالة الأساسية: `StrLen`

## ما وظيفة الكود؟

تحسب عدد runes في النص باستخدام range، لذلك العربية أو الإيموجي تُحسب كحروف Unicode لا كعدد بايتات UTF-8.

## الكود الأصلي

```go
package piscine

func StrLen(s string) int {
	count := 0

	for range s {
		count++
	}

	return count
}
```

## المدخلات والمخرجات

```go
func StrLen(s string) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. بدء count=0.
2. لكل rune داخل s زيادة count.
3. إرجاع count.

## تتبع بأمثلة

- `StrLen("Hello")` ⟶ `5`
- `StrLen("مرحبا")` ⟶ `5`

## المفاهيم المهمة

- **الفرق بين len(string) وعدد runes**
- **range بلا متغيرات**

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package piscine` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func StrLen(s string) int {` | تعريف دالة `StrLen` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `count := 0` | إنشاء المتغير `count` لأول مرة واستنتاج نوعه تلقائيًا من القيمة الموجودة يمين `:=`. |
| 6 | `for range s {` | حلقة `range` تمر على عناصر النص أو الـslice؛ المتغير `_` يعني تجاهل قيمة لا نحتاجها. |
| 7 | `count++` | يزيد قيمة العداد بمقدار 1. |
| 8 | `}` | نهاية الكتلة الحالية. |
| 10 | `return count` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 11 | `}` | نهاية الكتلة الحالية. |

## الحالات الخاصة والأخطاء الشائعة

- بعض الإيموجي المركبة قد تتكون من عدة runes رغم ظهورها كرمز بصري واحد.

## التعقيد

الزمن O(bytes in s) والذاكرة O(1).

## طريقة تجربة الدالة

لأن الملف يستخدم `package piscine` فهو ليس برنامجًا مستقلًا. أنشئ ملف `main.go` مؤقتًا في مجلد آخر يستورد الحزمة، أو استخدم ملف اختبار Go. مثال اختبار مبسط:

```go
package piscine

import "testing"

func TestStrLen(t *testing.T) {
	// ضع هنا حالات الاختبار المناسبة كما في الأمثلة أعلاه.
}
```

> لا تضف ملف `main.go` داخل مجلد التسليم إذا كانت منصة التمرين تطلب ملف الدالة فقط.
