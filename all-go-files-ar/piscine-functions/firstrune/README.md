# شرح `firstrune.go` — إرجاع أول rune من النص

> الدالة الأساسية: `FirstRune`

## ما وظيفة الكود؟

تعيد أول حرف Unicode من string، وليس أول byte فقط. إذا كان النص فارغًا تعيد rune صفرية.

## الكود الأصلي

```go
package piscine

func FirstRune(s string) rune {
	for _, r := range s {
		return r
	}

	return 0
}
```

## المدخلات والمخرجات

```go
func FirstRune(s string) rune
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. استخدام range على النص.
2. إرجاع أول rune فورًا من أول دورة.
3. إذا لم تبدأ الحلقة، إرجاع 0.

## تتبع بأمثلة

- `FirstRune("Hello")` ⟶ `'H'`
- `FirstRune("مرحبا")` ⟶ `'م'`
- `FirstRune("")` ⟶ `0`

## المفاهيم المهمة

- **UTF-8**
- **rune**
- **الإرجاع من أول دورة**

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package piscine` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func FirstRune(s string) rune {` | تعريف دالة `FirstRune` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `for _, r := range s {` | حلقة `range` تمر على عناصر النص أو الـslice؛ المتغير `_` يعني تجاهل قيمة لا نحتاجها. |
| 5 | `return r` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 6 | `}` | نهاية الكتلة الحالية. |
| 8 | `return 0` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 9 | `}` | نهاية الكتلة الحالية. |

## الحالات الخاصة والأخطاء الشائعة

- 0 ليست الحرف "0" بل القيمة الصفرية للنوع rune.

## التعقيد

الزمن O(1) بالنسبة لأول rune، والذاكرة O(1).

## طريقة تجربة الدالة

لأن الملف يستخدم `package piscine` فهو ليس برنامجًا مستقلًا. أنشئ ملف `main.go` مؤقتًا في مجلد آخر يستورد الحزمة، أو استخدم ملف اختبار Go. مثال اختبار مبسط:

```go
package piscine

import "testing"

func TestFirstRune(t *testing.T) {
	// ضع هنا حالات الاختبار المناسبة كما في الأمثلة أعلاه.
}
```

> لا تضف ملف `main.go` داخل مجلد التسليم إذا كانت منصة التمرين تطلب ملف الدالة فقط.
