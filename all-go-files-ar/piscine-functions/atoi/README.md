# شرح `atoi.go` — تحويل string إلى int مع الإشارة

> الدالة الأساسية: `Atoi`

## ما وظيفة الكود؟

تحول نصًا رقميًا صالحًا إلى عدد صحيح، وتدعم علامة + أو - في بداية النص. إذا وُجد حرف غير رقمي أو كانت الإشارة وحدها بلا أرقام تعيد 0.

## الكود الأصلي

```go
package piscine

func Atoi(s string) int {
	if s == "" {
		return 0
	}

	sign := 1
	start := 0

	if s[0] == '-' {
		sign = -1
		start = 1
	} else if s[0] == '+' {
		start = 1
	}

	if start == len(s) {
		return 0
	}

	result := 0

	for i := start; i < len(s); i++ {
		if s[i] < '0' || s[i] > '9' {
			return 0
		}

		result = result*10 + int(s[i]-'0')
	}

	return result * sign
}
```

## تدفق التنفيذ خطوة بخطوة

1. رفض النص الفارغ.
2. تحديد الإشارة ونقطة بداية قراءة الأرقام.
3. التأكد من وجود أرقام بعد الإشارة.
4. بناء العدد رقمًا بعد رقم بالمعادلة result = result*10 + digit.
5. ضرب الناتج في الإشارة قبل الإرجاع.

## تتبع بأمثلة

- `Atoi("123")` ⟶ `123`
- `Atoi("-42")` ⟶ `-42`
- `Atoi("+17")` ⟶ `17`
- `Atoi("12a")` ⟶ `0`

## المفاهيم المهمة

- **الفهرسة داخل string**
- **تحويل الحرف الرقمي إلى قيمته باستخدام s[i]-'0'**
- **الإشارة sign**

## الحالات الخاصة والأخطاء الشائعة

- النص الفارغ يعيد 0.
- علامة + أو - وحدها تعيد 0.
- أي محرف غير رقمي بعد العلامة يعيد 0.
- هذا الحل لا يحتوي حماية صريحة من overflow.

## التعقيد

الزمن O(n) والذاكرة O(1).
