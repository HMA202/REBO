# شرح `printcombn.go` — طباعة تركيبات n أرقام متزايدة

> الدالة الأساسية: `PrintCombN`

## ما وظيفة الكود؟

تعمم PrintComb لأي n: تطبع التركيبات الفريدة من أرقام 0..9 بطول n وبترتيب تصاعدي باستخدام recursion.

## الكود الأصلي

```go
package piscine

import "github.com/01-edu/z01"

func PrintCombN(n int) {
	comb := make([]int, n)
	first := true

	var loop func(pos int, start int)

	loop = func(pos int, start int) {
		if pos == n {
			if !first {
				z01.PrintRune(',')
				z01.PrintRune(' ')
			}

			for i := 0; i < n; i++ {
				z01.PrintRune('0' + rune(comb[i]))
			}

			first = false
			return
		}

		for i := start; i <= 10-n+pos; i++ {
			comb[pos] = i
			loop(pos+1, i+1)
		}
	}

	loop(0, 0)
	z01.PrintRune('\n')
}
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. إنشاء slice بطول n لتخزين التركيبة الحالية.
2. تعريف دالة recursive داخلية تأخذ الموضع وأقل رقم مسموح.
3. حالة الأساس pos==n تعني اكتمال التركيبة فتُطبع.
4. الحلقة تختار الرقم الحالي من start حتى الحد الذي يترك أرقامًا كافية لبقية المواضع.
5. الاستدعاء التالي يستخدم i+1 لضمان تزايد الأرقام وعدم تكرارها.

## تتبع بأمثلة

- `PrintCombN(2)` ⟶ `01, 02, ... 89`
- `PrintCombN(3)` ⟶ `012, 013, ... 789`

## المفاهيم المهمة

- **closure**
- **recursion/backtracking**
- **make([]int,n)**
- **حساب حدود الحلقة**

## الحالات الخاصة والأخطاء الشائعة

- التمرين عادة يختبر n من 1 إلى 9.
- first يمنع الفاصل قبل أول نتيجة.

## التعقيد

عدد النتائج C(10,n)، ووقت الطباعة O(n×C(10,n)).
