# شرح `index.go` — البحث عن نص فرعي

> الدالة الأساسية: `Index`

## ما وظيفة الكود؟

تعيد index لأول ظهور لـ toFind داخل s، أو -1 إذا لم يوجد. النص المطلوب الفارغ يعيد 0.

## الكود الأصلي

```go
package piscine

func Index(s string, toFind string) int {
	if toFind == "" {
		return 0
	}

	for i := 0; i+len(toFind) <= len(s); i++ {
		if s[i:i+len(toFind)] == toFind {
			return i
		}
	}

	return -1
}
```

## المدخلات والمخرجات

```go

```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. معالجة toFind الفارغ.
2. تحريك نقطة البداية i من 0 حتى آخر مكان يمكن أن يتسع للنص المطلوب.
3. أخذ المقطع s[i:i+len(toFind)] ومقارنته بـ toFind.
4. إرجاع أول index مطابق، وإلا -1.

## تتبع بأمثلة

- `Index("Hello World", "World")` ⟶ `6`
- `Index("banana", "na")` ⟶ `2`
- `Index("abc", "x")` ⟶ `-1`

## الحالات الخاصة والأخطاء الشائعة

- الفهارس هنا بايتية لا rune؛ لذلك index للنصوص Unicode هو byte index.
- النص المطلوب الأطول من s يؤدي مباشرة إلى عدم دخول الحلقة.

## التعقيد

الزمن O((n-m+1)×m) في أسوأ حالة والذاكرة O(1).
