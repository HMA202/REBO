# شرح `printcomb.go` — طباعة جميع تركيبات ثلاثة أرقام متزايدة

> الدالة الأساسية: `PrintComb`

## ما وظيفة الكود؟

تطبع جميع المجموعات المكونة من ثلاثة أرقام مختلفة مرتبة تصاعديًا، من 012 حتى 789، مع فاصلة ومسافة بين النتائج.

## الكود الأصلي

```go
package piscine

import "github.com/01-edu/z01"

func PrintComb() {
	for a := '0'; a <= '7'; a++ {
		for b := a + 1; b <= '8'; b++ {
			for c := b + 1; c <= '9'; c++ {
				z01.PrintRune(a)
				z01.PrintRune(b)
				z01.PrintRune(c)

				if !(a == '7' && b == '8' && c == '9') {
					z01.PrintRune(',')
					z01.PrintRune(' ')
				}
			}
		}
	}
	z01.PrintRune('\n')
}
```

## المدخلات والمخرجات


الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. الحلقة a تبدأ من 0 حتى 7.
2. b تبدأ دائمًا من a+1 لضمان الاختلاف والترتيب.
3. c تبدأ من b+1.
4. طباعة الثلاثة عند كل تركيب.
5. عدم طباعة الفاصل بعد آخر تركيب 789.

## تتبع بأمثلة

- `بداية الناتج` ⟶ `012, 013, 014, ...`
- `نهاية الناتج` ⟶ `..., 689, 789`

## المفاهيم المهمة

- **حلقات متداخلة**
- **rune رقمية**
- **منع delimiter أخير**
