# شرح `isalpha.go` — التحقق من أن النص أبجدي رقمي

> الدالة الأساسية: `IsAlpha`

## ما وظيفة الكود؟

تعيد true إذا كانت جميع المحارف حروفًا إنجليزية أو أرقامًا. أي مسافة أو رمز أو حرف خارج تلك المجالات يجعلها false.

## الكود الأصلي

```go
package piscine

func IsAlpha(s string) bool {
	for _, r := range s {
		isLower := r >= 'a' && r <= 'z'
		isUpper := r >= 'A' && r <= 'Z'
		isDigit := r >= '0' && r <= '9'

		if !isLower && !isUpper && !isDigit {
			return false
		}
	}

	return true
}
```


الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. المرور على كل rune.
2. تكوين ثلاث حالات: حرف صغير، حرف كبير، رقم.
3. إذا لم يتحقق أي منها إرجاع false فورًا.
4. بعد اكتمال الحلقة إرجاع true.

## تتبع بأمثلة

- `IsAlpha("Hello2026")` ⟶ `true`
- `IsAlpha("Hello 2026")` ⟶ `false`
- `IsAlpha("")` ⟶ `true`
