# شرح `fibonacci.go` — حساب فيبوناتشي بالاستدعاء الذاتي

> الدالة الأساسية: `Fibonacci`

## ما وظيفة الكود؟

تعيد قيمة فيبوناتشي عند index: F(0)=0 وF(1)=1، وكل قيمة لاحقة مجموع القيمتين السابقتين. تعيد -1 للفهرس السالب.

## الكود الأصلي

```go
package piscine

func Fibonacci(index int) int {
	if index < 0 {
		return -1
	}

	if index == 0 {
		return 0
	}

	if index == 1 {
		return 1
	}

	return Fibonacci(index-1) + Fibonacci(index-2)
}
```

## المدخلات والمخرجات

```go
func Fibonacci(index int) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. رفض index السالب.
2. إرجاع حالات الأساس 0 و1.
3. حساب Fibonacci(index-1) وFibonacci(index-2) وجمعهما.

## تتبع بأمثلة

- `Fibonacci(0)` ⟶ `0`
- `Fibonacci(1)` ⟶ `1`
- `Fibonacci(6)` ⟶ `8`
- `Fibonacci(-2)` ⟶ `-1`
