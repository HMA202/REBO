# شرح `isupper.go` — التحقق من الحروف الكبيرة

> الدالة الأساسية: `IsUpper`

## ما وظيفة الكود؟

تعيد true إذا كانت جميع محارف النص بين A وZ فقط.

## الكود الأصلي

```go
package piscine

func IsUpper(s string) bool {
	for _, r := range s {
		if r < 'A' || r > 'Z' {
			return false
		}
	}

	return true
}
```

## المدخلات والمخرجات

```go
func IsUpper(s string) bool
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. المرور على كل rune.
2. رفض أي rune خارج A..Z.
3. إرجاع true بعد الفحص.

## تتبع بأمثلة

- `IsUpper("GO")` ⟶ `true`
- `IsUpper("Go")` ⟶ `false`
- `IsUpper("")` ⟶ `true`

## المفاهيم المهمة

- **مجال ASCII**
- **الإرجاع المبكر**

## الحالات الخاصة والأخطاء الشائعة

- النص الفارغ يعيد true.
- الأرقام والمسافات غير مسموحة.

الزمن O(n) والذاكرة O(1).
