# شرح `isprintable.go` — التحقق من محارف ASCII القابلة للطباعة

> الدالة الأساسية: `IsPrintable`

## ما وظيفة الكود؟

تعيد true إذا كانت كل المحارف بين المسافة ASCII 32 والرمز ~ ASCII 126.

## الكود الأصلي

```go
package piscine

func IsPrintable(s string) bool {
	for _, r := range s {
		if r < ' ' || r > '~' {
			return false
		}
	}

	return true
}
```

## المدخلات والمخرجات

```go
func IsPrintable(s string) bool
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. المرور على كل rune.
2. رفض المحارف الأقل من المسافة أو الأكبر من ~.
3. إرجاع true عند اكتمال الفحص.

## تتبع بأمثلة

- `IsPrintable("Hello!")` ⟶ `true`
- `IsPrintable("Hello\n")` ⟶ `false`

## الحالات الخاصة والأخطاء الشائعة

- النص الفارغ يعيد true.
- الأحرف Unicode خارج ASCII تعطي false في هذا الحل.

## التعقيد

الزمن O(n) والذاكرة O(1).
