# شرح `basicatoi.go` — تحويل رقمي بسيط من string إلى int

> الدالة الأساسية: `BasicAtoi`

## ما وظيفة الكود؟

تبني عددًا صحيحًا من سلسلة أرقام بافتراض أن المدخل صالح بالكامل، من دون التحقق من الإشارة أو المحارف غير الرقمية.

## الكود الأصلي

```go
package piscine

func BasicAtoi(s string) int {
	result := 0

	for _, r := range s {
		result = result*10 + int(r-'0')
	}

	return result
}
```

## المدخلات والمخرجات

```go
func BasicAtoi(s string) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. بدء result بصفر.
2. المرور على كل rune في النص.
3. تحويل rune الرقمي إلى قيمة بين 0 و9 بطرح "0".
4. إزاحة العدد السابق منزلة عشرية ثم إضافة الرقم الجديد.

## تتبع بأمثلة

- `BasicAtoi("123")` ⟶ `123`
- `BasicAtoi("007")` ⟶ `7`

## المفاهيم المهمة

- **البناء التراكمي للأعداد**
- **طرح rune من rune**
- **range**

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package piscine` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func BasicAtoi(s string) int {` | تعريف دالة `BasicAtoi` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `result := 0` | إنشاء المتغير `result` لأول مرة واستنتاج نوعه تلقائيًا من القيمة الموجودة يمين `:=`. |
| 6 | `for _, r := range s {` | حلقة `range` تمر على عناصر النص أو الـslice؛ المتغير `_` يعني تجاهل قيمة لا نحتاجها. |
| 7 | `result = result*10 + int(r-'0')` | إسناد أو تحديث قيمة `result` وفق التعبير الموجود في الجهة اليمنى. |
| 8 | `}` | نهاية الكتلة الحالية. |
| 10 | `return result` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 11 | `}` | نهاية الكتلة الحالية. |

## الحالات الخاصة والأخطاء الشائعة

- التمرين يفترض أن النص يحتوي أرقامًا فقط.
- النص الفارغ يعيد 0 لأن الحلقة لا تعمل.
- لا يدعم + أو -.

## التعقيد

الزمن O(n) والذاكرة O(1).
