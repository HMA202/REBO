# شرح `join.go` — دمج النصوص مع فاصل

> الدالة الأساسية: `Join`

## ما وظيفة الكود؟

تجمع عناصر strs داخل string واحدة، وتضع sep بين كل عنصرين من دون وضعه قبل أول عنصر أو بعد الأخير.

## الكود الأصلي

```go
package piscine

func Join(strs []string, sep string) string {
	result := ""

	for i, str := range strs {
		if i > 0 {
			result += sep
		}

		result += str
	}

	return result
}
```

## المدخلات والمخرجات

```go
func Join(strs []string, sep string) string
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. بدء result فارغ.
2. المرور على العناصر مع index.
3. إذا لم يكن العنصر الأول، إضافة الفاصل.
4. إضافة العنصر نفسه.
5. إرجاع النتيجة.

## تتبع بأمثلة

- `Join([]string{"a","b","c"}, "-")` ⟶ `"a-b-c"`
- `Join([]string{}, ",")` ⟶ `""`

## المفاهيم المهمة

- **range مع index**
- **منع الفاصل الزائد**

## الحالات الخاصة والأخطاء الشائعة

- slice فارغة تعيد string فارغة.
- فاصل فارغ يجعلها مثل BasicJoin.

## التعقيد

يمر على كل النصوص؛ += قد يسبب نسخًا متكررًا، والذاكرة النهائية O(total length).
