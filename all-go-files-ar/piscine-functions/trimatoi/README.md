# شرح `trimatoi.go` — استخراج الأرقام من نص وبناء عدد

> الدالة الأساسية: `TrimAtoi`

## ما وظيفة الكود؟

تجمع كل الأرقام التي تظهر في النص بالترتيب وتتجاهل باقي المحارف. إذا ظهرت - قبل أول رقم تجعل النتيجة سالبة.

## الكود الأصلي

```go
package piscine

func TrimAtoi(s string) int {
	result := 0
	sign := 1
	foundDigit := false

	for _, r := range s {
		if r == '-' && !foundDigit {
			sign = -1
		}

		if r >= '0' && r <= '9' {
			result = result*10 + int(r-'0')
			foundDigit = true
		}
	}

	if !foundDigit {
		return 0
	}

	return result * sign
}
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. بدء result=0 وsign=1 وfoundDigit=false.
2. المرور على runes.
3. عند - قبل العثور على أي رقم، ضبط sign=-1.
4. عند الرقم، إضافته إلى النتيجة وتحديد foundDigit=true.
5. إذا لم يوجد رقم إرجاع 0، وإلا result*sign.

## تتبع بأمثلة

- `TrimAtoi("Hello! How are you? 42")` ⟶ `42`
- `TrimAtoi("-This is 4, you 2")` ⟶ `-42`
- `TrimAtoi("abc")` ⟶ `0`

## المفاهيم المهمة

- **فلترة المحارف**
- **بناء عدد**
- **حالة foundDigit**

## الحالات الخاصة والأخطاء الشائعة

- أي علامة - قبل أول رقم تجعل الناتج سالبًا، حتى لو ظهرت أكثر من مرة.
- علامة - بعد أول رقم تُتجاهل.
- علامة + لا تؤثر.

## التعقيد

الزمن O(n) والذاكرة O(1).
