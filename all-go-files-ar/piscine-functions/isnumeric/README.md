# شرح `isnumeric.go` — التحقق من أن النص أرقام فقط

> الدالة الأساسية: `IsNumeric`

## ما وظيفة الكود؟

تعيد true إذا كانت جميع المحارف أرقامًا من 0 إلى 9.

## الكود الأصلي

```go
package piscine

func IsNumeric(s string) bool {
	for _, r := range s {
		if r < '0' || r > '9' {
			return false
		}
	}

	return true
}
```

## المدخلات والمخرجات

```go
func IsNumeric(s string) bool
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. المرور على كل rune.
2. رفض أي rune خارج المجال الرقمي.
3. إرجاع true عند اكتمال الفحص.

## تتبع بأمثلة

- `IsNumeric("2026")` ⟶ `true`
- `IsNumeric("20.26")` ⟶ `false`
- `IsNumeric("")` ⟶ `true`

## المفاهيم المهمة

- **مجال الأرقام في ASCII**
- **الإرجاع المبكر**

## الحالات الخاصة والأخطاء الشائعة

- النص الفارغ يعيد true.
- الإشارة السالبة والنقطة ليستا أرقامًا.

## التعقيد

الزمن O(n) والذاكرة O(1).
