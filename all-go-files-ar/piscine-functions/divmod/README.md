# شرح `divmod.go` — القسمة والباقي باستخدام المؤشرات

> الدالة الأساسية: `DivMod`

## ما وظيفة الكود؟

تحسب خارج قسمة a على b وباقي القسمة، ثم تكتب النتيجتين داخل المتغيرين اللذين يشير إليهما div وmod.

## الكود الأصلي

```go
package piscine

func DivMod(a int, b int, div *int, mod *int) {
	*div = a / b
	*mod = a % b
}
```

## المدخلات والمخرجات

```go
func DivMod(a int, b int, div *int, mod *int)
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. حساب a / b وتخزينه في المكان المشار إليه بواسطة div.
2. حساب a % b وتخزينه في المكان المشار إليه بواسطة mod.

## تتبع بأمثلة

- `a=13, b=5 ثم DivMod(a,b,&div,&mod)` ⟶ `div=2, mod=3`

## المفاهيم المهمة

- **المؤشرات pointers**
- **فك الإشارة ***
- **القسمة الصحيحة والباقي**

## الحالات الخاصة والأخطاء الشائعة

- يجب ألا يكون b مساويًا للصفر وإلا يحدث panic.
- يجب أن تكون المؤشرات صالحة وغير nil.

## التعقيد

الزمن والذاكرة O(1).
