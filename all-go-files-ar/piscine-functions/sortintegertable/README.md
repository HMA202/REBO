# شرح `sortintegertable.go` — ترتيب slice أعداد تصاعديًا

> الدالة الأساسية: `SortIntegerTable`

## ما وظيفة الكود؟

ترتب الجدول داخل مكانه in-place بمقارنة كل عنصر مع العناصر التالية وتبديله مع أي قيمة أصغر.

## الكود الأصلي

```go
package piscine

func SortIntegerTable(table []int) {
	for i := 0; i < len(table)-1; i++ {
		for j := i + 1; j < len(table); j++ {
			if table[i] > table[j] {
				table[i], table[j] = table[j], table[i]
			}
		}
	}
}
```

## المدخلات والمخرجات

```go

```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. الحلقة الخارجية تثبت الموضع i.
2. الحلقة الداخلية تمر على كل j بعد i.
3. إذا table[i] أكبر من table[j] يتم التبديل.
4. بعد اكتمال الحلقة الداخلية يصبح أصغر عنصر متبقٍ في i.

## تتبع بأمثلة

- `[]int{5,2,4,1}` ⟶ `[]int{1,2,4,5}`

## التعقيد

الزمن O(n²) والذاكرة O(1).
