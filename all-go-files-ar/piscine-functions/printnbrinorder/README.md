# شرح `printnbrinorder.go` — طباعة خانات العدد مرتبة تصاعديًا

> الدالة الأساسية: `PrintNbrInOrder`

## ما وظيفة الكود؟

تحصي عدد مرات ظهور كل خانة في n ثم تطبع الخانات من 0 إلى 9 حسب تكرارها.

## الكود الأصلي

```go
package piscine

import "github.com/01-edu/z01"

func PrintNbrInOrder(n int) {
	digits := [10]int{}

	if n == 0 {
		z01.PrintRune('0')
		return
	}

	for n > 0 {
		digits[n%10]++
		n /= 10
	}

	for digit := 0; digit <= 9; digit++ {
		for digits[digit] > 0 {
			z01.PrintRune(rune('0' + digit))
			digits[digit]--
		}
	}
}
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. إنشاء مصفوفة counts بطول 10.
2. معالجة الصفر كحالة خاصة.
3. استخراج آخر خانة n%10 وزيادة عدادها، ثم حذفها بـ n/=10.
4. المرور من digit=0 إلى 9.
5. طباعة كل رقم بعدد مرات ظهوره.

## تتبع بأمثلة

- `PrintNbrInOrder(32145)` ⟶ `12345`
- `PrintNbrInOrder(321321)` ⟶ `112233`
- `PrintNbrInOrder(0)` ⟶ `0`
