# شرح `iterativepower.go` — حساب القوة بحلقة

> الدالة الأساسية: `IterativePower`

## ما وظيفة الكود؟

تحسب nb مرفوعًا إلى power عن طريق ضرب nb في النتيجة power مرات. القوة السالبة تعيد 0.

## الكود الأصلي

```go
package piscine

func IterativePower(nb int, power int) int {
	if power < 0 {
		return 0
	}

	result := 1

	for i := 0; i < power; i++ {
		result *= nb
	}

	return result
}
```

## المدخلات والمخرجات


الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. رفض power السالب.
2. بدء result=1 لأن أي عدد للقوة صفر يساوي 1.
3. تكرار الضرب power مرات.
4. إرجاع النتيجة.

## تتبع بأمثلة

- `IterativePower(2, 5)` ⟶ `32`
- `IterativePower(7, 0)` ⟶ `1`
- `IterativePower(2, -1)` ⟶ `0`

## الحالات الخاصة والأخطاء الشائعة

- لا توجد حماية من overflow.
- 0^0 يعيد 1 حسب حالة الأس صفر في هذا التمرين.

## التعقيد

الزمن O(power) والذاكرة O(1).
