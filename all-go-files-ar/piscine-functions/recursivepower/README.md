# شرح `recursivepower.go` — حساب القوة بالاستدعاء الذاتي

> الدالة الأساسية: `RecursivePower`

## ما وظيفة الكود؟

تحسب nb^power recursively، وتعالج القوة السالبة بإرجاع 0 والقوة صفر بإرجاع 1.

## الكود الأصلي

```go
package piscine

func RecursivePower(nb int, power int) int {
	if power < 0 {
		return 0
	}

	if power == 0 {
		return 1
	}

	return nb * RecursivePower(nb, power-1)
}
```

## المدخلات والمخرجات

```go

```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. رفض القوة السالبة.
2. حالة الأساس power==0 تعيد 1.
3. ضرب nb في نتيجة القوة الأقل بواحد.

## تتبع بأمثلة

- `RecursivePower(3,4)` ⟶ `81`
- `RecursivePower(5,0)` ⟶ `1`
