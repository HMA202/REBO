# شرح `iterativefactorial.go` — حساب العامل factorial بحلقة

> الدالة الأساسية: `IterativeFactorial`

## ما وظيفة الكود؟

تحسب nb! تكراريًا. تعيد 0 للسالب، وتحاول منع overflow بفحص النتيجة قبل الضرب.

## الكود الأصلي

```go
package piscine

func IterativeFactorial(nb int) int {
	if nb < 0 {
		return 0
	}

	result := 1
	maxInt := int(^uint(0) >> 1)

	for i := 2; i <= nb; i++ {
		if result > maxInt/i {
			return 0
		}

		result *= i
	}

	return result
}
```

## تدفق التنفيذ خطوة بخطوة

1. رفض nb السالب.
2. بدء result=1 لأن 0! و1! يساويان 1.
3. حساب أكبر int متاح في النظام.
4. الضرب في الأعداد من 2 إلى nb.
5. قبل كل ضرب، فحص result > maxInt/i؛ عندها الضرب سيتجاوز المجال فتعيد 0.

## تتبع بأمثلة

- `IterativeFactorial(5)` ⟶ `120`
- `IterativeFactorial(0)` ⟶ `1`
- `IterativeFactorial(-1)` ⟶ `0`
