# شرح `compare.go` — مقارنة نصين

> الدالة الأساسية: `Compare`

## ما وظيفة الكود؟

تعيد 0 إذا كان النصان متساويين، و-1 إذا كان a يأتي قبل b، و1 إذا كان a يأتي بعد b وفق المقارنة المعجمية في Go.

## الكود الأصلي

```go
package piscine

func Compare(a, b string) int {
	if a == b {
		return 0
	}

	if a < b {
		return -1
	}

	return 1
}
```

## المدخلات والمخرجات

```go
func Compare(a, b string) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. فحص التطابق أولًا.
2. إذا لم يتطابقا، فحص a < b.
3. إرجاع 1 للحالة المتبقية.

## تتبع بأمثلة

- `Compare("abc", "abc")` ⟶ `0`
- `Compare("abc", "abd")` ⟶ `-1`
- `Compare("z", "a")` ⟶ `1`

## المفاهيم المهمة

- **المقارنة المعجمية للـ string**
- **الإرجاع المبكر**

## الحالات الخاصة والأخطاء الشائعة

- الحروف الكبيرة والصغيرة لها ترتيب مختلف.
- المقارنة في Go تعتمد على بايتات UTF-8 عند مقارنة strings.

## التعقيد

في أسوأ حالة الزمن O(min(n,m)) تقريبًا، والذاكرة O(1).
