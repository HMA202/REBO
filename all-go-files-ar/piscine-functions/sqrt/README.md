# شرح `sqrt.go` — الجذر التربيعي الصحيح

> الدالة الأساسية: `Sqrt`

## ما وظيفة الكود؟

تعيد الجذر إذا كان nb مربعًا كاملًا، وإلا تعيد 0. القيم غير الموجبة تعيد 0 حسب هذا الحل.

## الكود الأصلي

```go
package piscine

func Sqrt(nb int) int {
	if nb <= 0 {
		return 0
	}

	for i := 1; i <= nb/i; i++ {
		if i*i == nb {
			return i
		}
	}

	return 0
}
```

## المدخلات والمخرجات

```go
func Sqrt(nb int) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. رفض nb<=0.
2. تجربة i بدءًا من 1.
3. الاستمرار ما دام i <= nb/i، وهو بديل آمن عن i*i<=nb.
4. إذا i*i==nb إرجاع i.
5. إرجاع 0 إذا لم توجد مطابقة.

## تتبع بأمثلة

- `Sqrt(25)` ⟶ `5`
- `Sqrt(26)` ⟶ `0`
- `Sqrt(0)` ⟶ `0`

## المفاهيم المهمة

- **المربع الكامل**
- **تجنب overflow في شرط الحلقة**

## الحالات الخاصة والأخطاء الشائعة

- الجذر 0 رياضيًا يساوي 0، والنتيجة نفسها هنا لكن عبر حالة الرفض.
- الأعداد السالبة تعيد 0.

## التعقيد

الزمن O(√n) والذاكرة O(1).
