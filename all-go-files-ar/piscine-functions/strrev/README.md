# شرح `strrev.go` — عكس النص مع دعم Unicode

> الدالة الأساسية: `StrRev`

## ما وظيفة الكود؟

تحول string إلى []rune ثم تبدل الحروف من الطرفين حتى المنتصف، وبعدها تعيدها string.

## الكود الأصلي

```go
package piscine

func StrRev(s string) string {
	runes := []rune(s)

	for i, j := 0, len(runes)-1; i < j; i, j = i+1, j-1 {
		runes[i], runes[j] = runes[j], runes[i]
	}

	return string(runes)
}
```

## المدخلات والمخرجات

```go
func StrRev(s string) string
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. تحويل النص إلى []rune حتى لا تنكسر أحرف UTF-8.
2. تعريف i من البداية وj من النهاية.
3. تبديل runes[i] وrunes[j].
4. تحريك i للأمام وj للخلف.
5. التحويل إلى string بعد انتهاء التبديل.

## تتبع بأمثلة

- `StrRev("Hello")` ⟶ `"olleH"`
- `StrRev("مرحبا")` ⟶ `"ابحرم"`

## المفاهيم المهمة

- **two pointers**
- **تبديل متزامن**
- **[]rune**
