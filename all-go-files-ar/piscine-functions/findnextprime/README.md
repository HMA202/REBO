# شرح `findnextprime.go` — إيجاد أول عدد أولي أكبر من أو يساوي العدد

> الدالة الأساسية: `FindNextPrime`

## ما وظيفة الكود؟

تعيد أصغر عدد أولي لا يقل عن nb، مع تجاوز الأعداد الزوجية لتقليل الفحوص.

## الكود الأصلي

```go
package piscine

func FindNextPrime(nb int) int {
	if nb <= 2 {
		return 2
	}

	if nb%2 == 0 {
		nb++
	}

	for !isPrimeNext(nb) {
		nb += 2
	}

	return nb
}

func isPrimeNext(nb int) bool {
	if nb < 2 {
		return false
	}

	if nb == 2 {
		return true
	}

	if nb%2 == 0 {
		return false
	}

	for divisor := 3; divisor <= nb/divisor; divisor += 2 {
		if nb%divisor == 0 {
			return false
		}
	}

	return true
}
```

## تدفق التنفيذ خطوة بخطوة

1. كل قيمة <=2 تعيد 2.
2. إذا كان العدد زوجيًا، تحويله إلى الفردي التالي.
3. فحص الأعداد الفردية واحدًا بعد آخر بزيادة 2.
4. استخدام isPrimeNext لفحص القواسم الفردية حتى الجذر التربيعي بطريقة divisor <= nb/divisor.

## تتبع بأمثلة

- `FindNextPrime(14)` ⟶ `17`
- `FindNextPrime(17)` ⟶ `17`
- `FindNextPrime(-5)` ⟶ `2`

## الحالات الخاصة والأخطاء الشائعة

- القيم السالبة والصفر و1 تعيد 2.
- تجنب divisor*divisor يقلل احتمال overflow.

## التعقيد

لكل مرشح فحص حتى O(√n)، وقد تُفحص عدة مرشحات؛ الذاكرة O(1).
