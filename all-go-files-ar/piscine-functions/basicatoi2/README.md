# شرح `basicatoi2.go` — تحويل string رقمي مع التحقق

> الدالة الأساسية: `BasicAtoi2`

## ما وظيفة الكود؟

تشبه BasicAtoi، لكنها تتحقق أن كل حرف رقم من 0 إلى 9؛ عند أول محرف غير رقمي تعيد 0.

## الكود الأصلي

```go
package piscine

func BasicAtoi2(s string) int {
	result := 0

	for _, r := range s {
		if r < '0' || r > '9' {
			return 0
		}

		result = result*10 + int(r-'0')
	}

	return result
}
```

## المدخلات والمخرجات

```go
func BasicAtoi2(s string) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. بدء result بصفر.
2. المرور على كل rune.
3. رفض الحرف إن كان خارج المجال من 0 إلى 9.
4. بناء العدد منزلة بعد منزلة.
5. إرجاع الناتج بعد اكتمال الفحص.

## تتبع بأمثلة

- `BasicAtoi2("2048")` ⟶ `2048`
- `BasicAtoi2("12x")` ⟶ `0`

## المفاهيم المهمة

- **التحقق المبكر**
- **تحويل rune رقمي إلى int**

## الحالات الخاصة والأخطاء الشائعة

- لا تدعم الإشارات.
- النص الفارغ يعيد 0.
- أي محرف غير رقمي يلغي العملية كلها.

## التعقيد

الزمن O(n) والذاكرة O(1).
