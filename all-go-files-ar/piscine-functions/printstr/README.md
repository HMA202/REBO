# شرح `printstr.go` — طباعة string حرفًا حرفًا

> الدالة الأساسية: `PrintStr`

## ما وظيفة الكود؟

تطبع كل rune في النص باستخدام z01.PrintRune.

## الكود الأصلي

```go
package piscine

import "github.com/01-edu/z01"

func PrintStr(s string) {
	for _, r := range s {
		z01.PrintRune(r)
	}
}
```

## المدخلات والمخرجات

```go
func PrintStr(s string)
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. المرور على النص باستخدام range.
2. إرسال كل rune إلى PrintRune.

## تتبع بأمثلة

- `PrintStr("Hello")` ⟶ `Hello`

## المفاهيم المهمة

- **range**
- **rune**
- **Unicode**

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package piscine` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `import "github.com/01-edu/z01"` | يستورد الحزمة المذكورة حتى يمكن استخدام دوالها أو قيمها داخل الملف. |
| 5 | `func PrintStr(s string) {` | تعريف دالة `PrintStr` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 6 | `for _, r := range s {` | حلقة `range` تمر على عناصر النص أو الـslice؛ المتغير `_` يعني تجاهل قيمة لا نحتاجها. |
| 7 | `z01.PrintRune(r)` | يطبع rune واحدة باستخدام الدالة المسموحة في تمارين piscine. |
| 8 | `}` | نهاية الكتلة الحالية. |
| 9 | `}` | نهاية الكتلة الحالية. |

## الحالات الخاصة والأخطاء الشائعة

- لا تضيف newline.
- النص الفارغ لا يطبع شيئًا.

## التعقيد

الزمن O(n) والذاكرة O(1).
