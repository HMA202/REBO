# شرح `tolower.go` — تحويل الحروف الإنجليزية إلى صغيرة

> الدالة الأساسية: `ToLower`

## ما وظيفة الكود؟

تمر على النص وتحول فقط الحروف A..Z إلى a..z، وتترك غيرها كما هو.

## الكود الأصلي

```go
package piscine

func ToLower(s string) string {
	result := ""

	for _, r := range s {
		if r >= 'A' && r <= 'Z' {
			r = r + ('a' - 'A')
		}

		result += string(r)
	}

	return result
}
```

## المدخلات والمخرجات

```go
func ToLower(s string) string
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. إنشاء result فارغ.
2. المرور على runes.
3. إذا كان الحرف كبيرًا إضافة فرق ASCII بين a وA.
4. إلحاق الحرف الناتج بـresult.

## تتبع بأمثلة

- `ToLower("Go 2026!")` ⟶ `"go 2026!"`

## المفاهيم المهمة

- **فارق ASCII**
- **تحويل rune إلى string**

## الحالات الخاصة والأخطاء الشائعة

- الأحرف غير الإنجليزية لا تتغير.
- استخدام += قد يسبب نسخًا متكررًا.

## التعقيد
