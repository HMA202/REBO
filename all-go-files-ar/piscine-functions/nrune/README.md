# شرح `nrune.go` — إرجاع الحرف رقم n

> الدالة الأساسية: `NRune`

## ما وظيفة الكود؟

تعيد الـ rune الموجودة في الموضع n مع اعتبار أول حرف في الموضع 1. تعيد 0 إذا كان n غير موجب أو أكبر من عدد الحروف.

## الكود الأصلي

```go
package piscine

func NRune(s string, n int) rune {
	if n <= 0 {
		return 0
	}

	position := 1

	for _, r := range s {
		if position == n {
			return r
		}

		position++
	}

	return 0
}
```

## المدخلات والمخرجات

```go
func NRune(s string, n int) rune
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. رفض n <= 0.
2. بدء position من 1.
3. المرور على runes النص.
4. عندما يساوي position قيمة n إرجاع الحرف.
5. زيادة الموضع، وإرجاع 0 إذا انتهى النص.

## تتبع بأمثلة

- `NRune("Hello", 1)` ⟶ `'H'`
- `NRune("Hello", 4)` ⟶ `'l'`
- `NRune("Hi", 5)` ⟶ `0`

## المفاهيم المهمة

- **ترقيم يبدأ من 1**
- **range على Unicode**

## الحالات الخاصة والأخطاء الشائعة

- n ليست index تبدأ من صفر؛ هي موضع يبدأ من 1.

## التعقيد

الزمن O(n) حتى الموضع المطلوب والذاكرة O(1).
