# شرح `ultimatedivmod.go` — القسمة والباقي داخل نفس المتغيرين

> الدالة الأساسية: `UltimateDivMod`

## ما وظيفة الكود؟

تقرأ القيمتين عبر pointer، ثم تستبدل a بخارج القسمة وb بالباقي. تستخدم متغيرين مؤقتين حتى لا تضيع القيمة الأصلية قبل الحساب الثاني.

## الكود الأصلي

```go
package piscine

func UltimateDivMod(a *int, b *int) {
	div := *a / *b
	mod := *a % *b

	*a = div
	*b = mod
}
```

## المدخلات والمخرجات

```go
func UltimateDivMod(a *int, b *int)
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. حساب div من القيمتين الأصليتين.
2. حساب mod قبل تعديل أي منهما.
3. كتابة div في *a وmod في *b.

## تتبع بأمثلة

- `a=13,b=5; UltimateDivMod(&a,&b)` ⟶ `a=2,b=3`

## المفاهيم المهمة

- **pointers**
- **ضرورة المتغيرات المؤقتة**
- **القسمة والباقي**
