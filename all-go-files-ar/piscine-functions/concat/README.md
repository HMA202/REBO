# شرح `concat.go` — دمج نصين

> الدالة الأساسية: `Concat`

## ما وظيفة الكود؟

تعيد نصًا جديدًا مكوّنًا من str1 ثم str2 مباشرة.

## الكود الأصلي

```go
package piscine

func Concat(str1 string, str2 string) string {
	return str1 + str2
}
```

## المدخلات والمخرجات

```go
func Concat(str1 string, str2 string) string
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. استخدام عامل الجمع + بين النصوص.
2. إرجاع النتيجة مباشرة.

## تتبع بأمثلة

- `Concat("Hello", "World")` ⟶ `"HelloWorld"`

## المفاهيم المهمة

- **دمج strings**
- **دالة ذات return مباشر**

## شرح كل سطر

| السطر | التعليمة | الشرح |
|---:|---|---|
| 1 | `package piscine` | يحدد الحزمة التي ينتمي إليها الملف؛ `piscine` تعني أن الدالة جزء من مكتبة التمارين، و`main` تعني برنامجًا قابلًا للتشغيل. |
| 3 | `func Concat(str1 string, str2 string) string {` | تعريف دالة `Concat` مع المعاملات ونوع الإرجاع الظاهرين في التوقيع. |
| 4 | `return str1 + str2` | تنهي الدالة فورًا وتعيد القيمة أو التعبير المذكور إلى المستدعي. |
| 5 | `}` | نهاية الكتلة الحالية. |

## الحالات الخاصة والأخطاء الشائعة

- إذا كان أحد النصين فارغًا تعيد النص الآخر.
- إذا كانا فارغين تعيد نصًا فارغًا.

## التعقيد

الزمن والذاكرة O(n+m) لإنشاء النص الجديد.

## طريقة تجربة الدالة

لأن الملف يستخدم `package piscine` فهو ليس برنامجًا مستقلًا. أنشئ ملف `main.go` مؤقتًا في مجلد آخر يستورد الحزمة، أو استخدم ملف اختبار Go. مثال اختبار مبسط:

```go
package piscine

import "testing"

func TestConcat(t *testing.T) {
	// ضع هنا حالات الاختبار المناسبة كما في الأمثلة أعلاه.
}
```

> لا تضف ملف `main.go` داخل مجلد التسليم إذا كانت منصة التمرين تطلب ملف الدالة فقط.
