# شرح `recursivefactorial.go` — حساب العامل بالاستدعاء الذاتي

> الدالة الأساسية: `RecursiveFactorial`

## ما وظيفة الكود؟

تحسب nb! recursively. تعيد 0 للسالب وللقيم فوق 20، وتعيد 1 عند الصفر.

## الكود الأصلي

```go
package piscine

func RecursiveFactorial(nb int) int {
	if nb < 0 || nb > 20 {
		return 0
	}

	if nb == 0 {
		return 1
	}

	return nb * RecursiveFactorial(nb-1)
}
```

## المدخلات والمخرجات

```go
func RecursiveFactorial(nb int) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. رفض القيم خارج 0..20.
2. حالة الأساس 0! = 1.
3. إرجاع nb مضروبًا في factorial للعدد السابق.

## تتبع بأمثلة

- `RecursiveFactorial(5)` ⟶ `120`
- `RecursiveFactorial(0)` ⟶ `1`
- `RecursiveFactorial(-1)` ⟶ `0`

- لا توجد حلقة.
