# شرح `alphacount.go` — عدّ الحروف الإنجليزية

> الدالة الأساسية: `AlphaCount`

## ما وظيفة الكود؟

تحسب الدالة عدد الحروف الإنجليزية فقط داخل النص، سواء كانت صغيرة من a إلى z أو كبيرة من A إلى Z، وتتجاهل الأرقام والمسافات والرموز.

## الكود الأصلي

```go
package piscine

func AlphaCount(s string) int {
	count := 0

	for _, r := range s {
		if (r >= 'a' && r <= 'z') || (r >= 'A' && r <= 'Z') {
			count++
		}
	}

	return count
}
```

## المدخلات والمخرجات

```go
func AlphaCount(s string) int
```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. إنشاء عدّاد يبدأ من الصفر.
2. المرور على النص حرفًا حرفًا باستخدام range حتى نتعامل مع كل rune على حدة.
3. فحص هل الحرف يقع داخل مجال الحروف الصغيرة أو الكبيرة في ASCII.
4. زيادة العداد عند تحقق الشرط، ثم إرجاع النتيجة النهائية.

## تتبع بأمثلة

- `AlphaCount("Hello 2026!")` ⟶ `5`
- `AlphaCount("Go-Lang")` ⟶ `6`
- `AlphaCount("123 !")` ⟶ `0`

## المفاهيم المهمة

- **rune وrange**
- **المقارنات على قيم ASCII**
- **العوامل المنطقية && و||**

## الحالات الخاصة والأخطاء الشائعة

- النص الفارغ يعيد 0.
- الأحرف العربية لا تُحسب لأن التمرين يقيد العد بالحروف اللاتينية الإنجليزية.
- الرقم أو المسافة أو الرمز لا يزيد العداد.

## التعقيد

الزمن O(n) والذاكرة الإضافية O(1).
