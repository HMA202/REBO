# شرح `printnbr.go` — طباعة int باستخدام PrintRune فقط

> الدالة الأساسية: `PrintNbr`

## ما وظيفة الكود؟

تطبع العدد الصحيح دون fmt، باستخدام recursion لاستخراج الخانات من اليسار إلى اليمين، مع معالجة الأعداد السالبة بطريقة تتجنب تحويل أصغر int إلى موجب.

## الكود الأصلي

```go
package piscine

import "github.com/01-edu/z01"

func PrintNbr(n int) {
	if n < 0 {
		z01.PrintRune('-')

		if n <= -10 {
			PrintNbr(-(n / 10))
		}

		digit := -(n % 10)
		r := '0'
		for i := 0; i < digit; i++ {
			r++
		}
		z01.PrintRune(r)
		return
	}

	if n >= 10 {
		PrintNbr(n / 10)
	}

	digit := n % 10
	r := '0'
	for i := 0; i < digit; i++ {
		r++
	}
	z01.PrintRune(r)
}
```

## المدخلات والمخرجات

```go

```

الدالة تستقبل القيم حسب التوقيع أعلاه، ثم إما تعيد نتيجة بواسطة `return` أو تعدّل قيمة عبر pointer أو تطبع باستخدام `z01.PrintRune`.

## تدفق التنفيذ خطوة بخطوة

1. إذا كان سالبًا، طباعة -.
2. للجزء السالب: استدعاء الدالة على -(n/10) للخانات السابقة ثم أخذ -(n%10) للخانة الأخيرة.
3. للعدد الموجب: استدعاء PrintNbr(n/10) إذا كان مكونًا من أكثر من خانة.
4. تحويل قيمة الخانة إلى rune بدءًا من 0 ثم زيادتها digit مرات.
5. طباعة rune الناتجة.

## تتبع بأمثلة

- `PrintNbr(42)` ⟶ `42`
- `PrintNbr(-305)` ⟶ `-305`
- `PrintNbr(0)` ⟶ `0`
