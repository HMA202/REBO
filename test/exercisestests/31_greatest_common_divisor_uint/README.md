# 31. Greatest Common Divisor Uint

## Task

Return the greatest number that divides both unsigned integers.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
1
12
25
```
