# 52. CSV Parse Line

## Task

Parse a simple CSV line. Supports commas inside double quotes.

## Files

```txt
exercise.go
main.go
README.md
```

## Run

```bash
go run .
```

## Expected Output

```txt
[Ali 25 Bahrain]
[Ali Manama, Bahrain 25]
```

