# 35. String To Int

## Task

Convert a string number into an int. Supports optional + or - sign.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
123
-45
90
0
```
