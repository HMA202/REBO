# 41. Decimal To Binary

## Task

Convert a positive decimal int into a binary string.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
1010
1111
0
```
