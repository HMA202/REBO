# 32. Least Common Multiple Uint

## Task

Return the smallest positive number that is divisible by both unsigned integers.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
517
24
36
```
