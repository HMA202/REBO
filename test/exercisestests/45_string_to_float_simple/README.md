# 45. String To Float Simple

## Task

Convert a simple decimal string into float64. Supports one dot and optional minus sign.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
12.5
-3.14
0
```
