# 44. Count Upper Lower

## Task

Return how many uppercase and lowercase English letters are in a string.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
6
4
```
