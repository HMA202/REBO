# 40. Binary To Decimal

## Task

Convert a binary string into a decimal int. Return 0 for invalid input.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
10
15
0
```
