# 42. Hex Digit Value

## Task

Return the decimal value of a hexadecimal digit. Return -1 if invalid.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
9
10
15
-1
```
