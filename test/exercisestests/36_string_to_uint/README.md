# 36. String To Uint

## Task

Convert a string number into uint. Return 0 if the string contains non-digits.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
123
0
99
0
```
