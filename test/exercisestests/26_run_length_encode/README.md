# 26. Run Length Encode

## Task

Compress repeated consecutive characters by writing count then character.

## Files

- `exercise.go` contains the function.
- `main.go` contains a simple test.

## Run

```bash
go run .
```

## Expected Output

```txt
1i 1l4o1v1e 1y1o1u
3a2b1c
```
